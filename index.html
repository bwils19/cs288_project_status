About This Project

A Natural Language Query System for Data Visualization.
Initial Goals of the Project:

  • Develop a sophisticated NLP system the can interpret complex, natural language queries about data.
  • Create an AI that can translate these queries into appropriate database queries and data manipulations.
  • Design an algorithm that selects and generates the most appropriate visualizations based on the query and data characteristics.
